Introduction to JavaScript

What is JavaScript?

JavaScript is a high-level, interpreted programming language that was created by Brendan Eich at Netscape in just 10 days in 1995. Originally designed to add interactivity to web pages, JavaScript has evolved into one of the most powerful and versatile programming languages in the world. Today it runs not only in browsers but also on servers (Node.js), mobile devices (React Native), desktops (Electron), and even embedded systems.

JavaScript is the only programming language that runs natively in web browsers. Every website you've ever interacted with — clicking buttons, submitting forms, seeing animations, loading content without refreshing — uses JavaScript behind the scenes. Together with HTML (structure) and CSS (styling), JavaScript forms the three pillars of web development.

Despite sharing a name with Java, the two languages are fundamentally different. JavaScript is dynamically typed, prototype-based, and interpreted, while Java is statically typed, class-based, and compiled. The name "JavaScript" was a marketing decision by Netscape to capitalize on Java's popularity at the time.

Modern JavaScript (ES6 and beyond) has introduced powerful features like arrow functions, template literals, destructuring, modules, promises, and async/await. These additions have transformed JavaScript from a simple scripting language into a robust, feature-rich language suitable for building complex applications at any scale.

💡

Fun Fact: JavaScript was created in just 10 days! Brendan Eich developed it in May 1995 under the codename "Mocha," which was later renamed to "LiveScript," and finally to "JavaScript." Despite its rapid creation, it became the most widely used programming language in history.

How JavaScript Works in the Browser

To use JavaScript in a web page, you can either write code directly in an HTML file using <script> tags, or link to an external .js file. The browser's JavaScript engine (V8 in Chrome, SpiderMonkey in Firefox) reads and executes the code. The easiest way to start experimenting is by opening your browser's Developer Tools (press F12) and typing JavaScript directly into the Console.

javascript 
// Inline JavaScript inside an HTML file
<script>
    // This runs when the browser loads the page
    console.log("Hello, World!");
    alert("Welcome to JavaScript!");
</script>

// Or link an external file (recommended)
<script src="app.js"></script>

Your First JavaScript Program

Let's write your first real JavaScript code. Create an HTML file and add a script that interacts with the page. JavaScript can write to the console, display alerts, modify the page content, and respond to user actions:

javascript 
// My first JavaScript program
console.log("Hello, World!");
console.log("Welcome to JavaScript!");

// Basic math operations
console.log("2 + 3 =", 2 + 3);       // 5
console.log("10 * 5 =", 10 * 5);     // 50
console.log("2 ** 8 =", 2 ** 8);     // 256 (exponentiation)

// Variables and string concatenation
let name = "CodeVerse";
console.log(`Welcome to ${name}!`);

// Get the current date
let now = new Date();
console.log("Current time:", now.toLocaleString());
📝

Note: console.log() is the primary way to output information in JavaScript. It prints to the browser's Developer Tools console (press F12 to open). Unlike alert(), it doesn't interrupt the user and is the standard tool for debugging.

The Browser Developer Console

Every modern browser includes Developer Tools (DevTools) with a built-in JavaScript console. This is your best friend while learning JavaScript. You can type any JavaScript expression and see the result instantly — just like a calculator, but far more powerful. Press F12 or Ctrl+Shift+J (Windows) / Cmd+Option+J (Mac) to open it.

javascript 
// Try these in your browser console (F12)
2 + 2                          // 4
"Hello".toUpperCase()          // "HELLO"
Math.random()                   // random number 0-1
typeof 42                       // "number"
typeof "hello"                  // "string"
[1, 2, 3].reverse()             // [3, 2, 1]
document.querySelector("h1")   // selects first h1 element
⚠

Important: Always use modern JavaScript (ES6+). Avoid the outdated var keyword for new code — use let and const instead. We'll explain the differences in the Variables lesson. Most modern browsers support ES6+ features, and tools like Babel can transpile for older browsers if needed.
