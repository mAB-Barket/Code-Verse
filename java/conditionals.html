Conditional Statements

if Statement

The if statement is the most basic form of decision making in Java. It evaluates a boolean expression inside parentheses, and if the condition is true, the code block inside the curly braces is executed. If the condition is false, the block is skipped entirely.

java 
int temperature = 35;

if (temperature > 30) {
    System.out.println("It's a hot day!");
    System.out.println("Stay hydrated.");
}

// Output:
// It's a hot day!
// Stay hydrated.

if-else Statement

The if-else statement adds an alternative path. When the condition is true, the first block runs; when it's false, the else block runs instead. This guarantees that exactly one of the two blocks will execute.

java 
int age = 16;

if (age >= 18) {
    System.out.println("You can vote.");
} else {
    System.out.println("You are too young to vote.");
}

// Output: You are too young to vote.

else-if Chain

When you have multiple conditions to test, use an else if chain. Java checks each condition in order from top to bottom. As soon as one condition is true, its block executes and the rest of the chain is skipped. The final else is optional and acts as a catch-all for any remaining cases.

java 
int score = 82;
char grade;

if (score >= 90) {
    grade = 'A';
} else if (score >= 80) {
    grade = 'B';
} else if (score >= 70) {
    grade = 'C';
} else if (score >= 60) {
    grade = 'D';
} else {
    grade = 'F';
}

System.out.println("Grade: " + grade);
// Output: Grade: B
💡

Tip: Order your else if conditions from most specific to most general. Placing a broad condition first may cause it to match before a more specific one gets a chance to be evaluated.

switch Statement

The switch statement provides a cleaner alternative to long else if chains when you need to compare a single variable against many possible values. It supports int, char, String, and enum types. Each case ends with a break to prevent fall-through, and default handles any unmatched values.

java 
String day = "Wednesday";

switch (day) {
    case "Monday":
        System.out.println("Start of the work week.");
        break;
    case "Wednesday":
        System.out.println("Midweek!");
        break;
    case "Friday":
        System.out.println("Almost weekend!");
        break;
    case "Saturday":
    case "Sunday":
        System.out.println("Weekend! Enjoy!");
        break;
    default:
        System.out.println("A regular day.");
        break;
}

// Output: Midweek!
⚠

Common Mistake: Forgetting the break keyword in a switch case causes fall-through — execution continues into the next case regardless of whether it matches. This is intentional in the Saturday/Sunday example above, but is usually a bug when unintended.

Ternary Operator

The ternary operator is a compact, one-line way to write simple if-else statements. It evaluates a condition and returns one of two values. The syntax is: condition ? valueIfTrue : valueIfFalse. It's great for assignments and inline expressions, but avoid nesting them as it hurts readability.

java 
int age = 20;

// Ternary operator
String status = (age >= 18) ? "Adult" : "Minor";
System.out.println(status);  // Adult

// Using ternary in a print statement
int score = 75;
System.out.println(score >= 60 ? "Pass" : "Fail");  // Pass

// Ternary to find the larger number
int a = 15, b = 22;
int max = (a > b) ? a : b;
System.out.println("Max: " + max);  // Max: 22
📝

Note: The ternary operator is an expression, not a statement — it returns a value. This means you can use it inside method arguments, string concatenation, and variable assignments. Keep it simple: if the logic is complex, use a regular if-else instead.
