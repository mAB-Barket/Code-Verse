Arrays & Collections

Declaring Arrays

An array in Java is a fixed-size container that holds elements of the same data type. Arrays are one of the most fundamental data structures and are stored in contiguous memory locations, which makes accessing elements by index very efficient. You must specify the type and size of an array when you create it.

There are several ways to declare and initialize arrays in Java. You can declare an array with values directly using an initializer list, or create an empty array of a specific size that you fill in later.

java 
// Declare and initialize with values
int[] numbers = {1, 2, 3, 4, 5};

// Declare an empty array of a specific size
String[] names = new String[3];
names[0] = "Alice";
names[1] = "Bob";
names[2] = "Charlie";

// Declare with explicit type on both sides
double[] prices = new double[]{9.99, 19.99, 29.99};

// Multi-dimensional array
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
💡

Tip: In Java, arrays are objects. When you create an array with new, numeric elements default to 0, boolean elements to false, and object references to null.

Accessing & Modifying

Array elements are accessed using their index, which starts at 0. The first element is at index 0, the second at index 1, and so on. You can read or update any element using bracket notation. The .length property gives you the total number of elements in the array.

java 
int[] scores = {85, 92, 78, 95, 88};

// Access elements by index
int first = scores[0];     // 85
int last = scores[4];      // 88

// Modify an element
scores[2] = 80;           // Change 78 to 80

// Get array length
int size = scores.length;   // 5

// Access the last element dynamically
int lastElement = scores[scores.length - 1];  // 88

System.out.println("First: " + first);
System.out.println("Array length: " + size);
⚠

Common Mistake: Accessing an index outside the array bounds (e.g., scores[5] on a 5-element array) throws an ArrayIndexOutOfBoundsException. Always ensure your index is between 0 and array.length - 1.

Iterating Arrays

There are two common ways to iterate over arrays in Java. The standard for loop gives you access to the index, which is useful when you need to know the position of each element. The enhanced for loop (also called "for-each") provides a cleaner syntax when you just need the values.

java 
String[] colors = {"Red", "Green", "Blue", "Yellow"};

// Standard for loop — gives index access
for (int i = 0; i < colors.length; i++) {
    System.out.println("Index " + i + ": " + colors[i]);
}

// Enhanced for loop (for-each) — cleaner syntax
for (String color : colors) {
    System.out.println(color);
}

// Output:
// Index 0: Red
// Index 1: Green
// Index 2: Blue
// Index 3: Yellow
// Red
// Green
// Blue
// Yellow
💡

Tip: Use the enhanced for loop when you don't need the index. It's less error-prone since you can't accidentally go out of bounds, and it reads more clearly.

ArrayList

While arrays have a fixed size, Java provides the ArrayList class for dynamic, resizable lists. ArrayList is part of the java.util package and can grow or shrink as elements are added or removed. Unlike arrays, ArrayList can only hold objects — not primitive types — so you use wrapper classes like Integer instead of int.

java 
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        // Create an ArrayList of Strings
        ArrayList<String> fruits = new ArrayList<>();

        // Add elements
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Cherry");

        // Access an element by index
        String first = fruits.get(0);  // "Apple"

        // Remove an element
        fruits.remove("Banana");

        // Get the size
        int count = fruits.size();    // 2

        // Iterate with for-each
        for (String fruit : fruits) {
            System.out.println(fruit);
        }

        // ArrayList of integers (must use Integer wrapper)
        ArrayList<Integer> nums = new ArrayList<>();
        nums.add(10);
        nums.add(20);
        nums.add(30);

        System.out.println("Size: " + nums.size());  // 3
    }
}
📝

Note: ArrayList uses .size() to get the number of elements, whereas arrays use .length (no parentheses). This is a common source of confusion for beginners.
