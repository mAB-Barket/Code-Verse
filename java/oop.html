Object-Oriented Programming

Classes and Objects

A class is a blueprint for creating objects. It defines properties (fields) and behaviors (methods) that the objects will have. An object is a specific instance of a class — it holds actual values and can perform actions defined by the class.

java 
// Define a class
public class Dog {
    // Fields (attributes)
    String name;
    int age;

    // Method
    public void bark() {
        System.out.println("Woof! My name is " + name);
    }
}

// Creating objects
Dog dog1 = new Dog();
dog1.name = "Buddy";
dog1.age = 3;
dog1.bark();  // Woof! My name is Buddy

Dog dog2 = new Dog();
dog2.name = "Max";
dog2.age = 5;
dog2.bark();  // Woof! My name is Max

Constructors

A constructor is a special method that is called automatically when an object is created. It initializes the object's fields. Java supports both default (no-argument) and parameterized constructors.

java 
public class Dog {
    String name;
    int age;

    // Default constructor
    public Dog() {
        this.name = "Unknown";
        this.age = 0;
    }

    // Parameterized constructor
    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public void info() {
        System.out.println(name + " is " + age + " years old.");
    }
}

// Using constructors
Dog dog1 = new Dog();                    // Uses default constructor
Dog dog2 = new Dog("Buddy", 3);          // Uses parameterized constructor
dog1.info();  // Unknown is 0 years old.
dog2.info();  // Buddy is 3 years old.

Encapsulation

Encapsulation means bundling data (fields) and the methods that operate on them into a single unit, while hiding the internal details from outside access. In Java, this is achieved by making fields private and providing public getter and setter methods.

java 
public class Student {
    // Private fields — cannot be accessed directly from outside
    private String name;
    private int age;

    // Constructor
    public Student(String name, int age) {
        this.name = name;
        setAge(age);  // Use setter for validation
    }

    // Getter for name
    public String getName() {
        return name;
    }

    // Setter for name
    public void setName(String name) {
        this.name = name;
    }

    // Getter for age
    public int getAge() {
        return age;
    }

    // Setter for age with validation
    public void setAge(int age) {
        if (age > 0 && age < 150) {
            this.age = age;
        }
    }
}

// Using encapsulated class
Student s = new Student("Alice", 20);
System.out.println(s.getName());  // Alice
s.setAge(21);
System.out.println(s.getAge());   // 21

Inheritance

Inheritance allows a class (child) to inherit fields and methods from another class (parent) using the extends keyword. The child class can reuse the parent's code and add its own features. Use super() to call the parent class constructor.

java 
// Parent class
public class Animal {
    String name;

    public Animal(String name) {
        this.name = name;
    }

    public void eat() {
        System.out.println(name + " is eating.");
    }

    public void speak() {
        System.out.println(name + " makes a sound.");
    }
}

// Child class — inherits from Animal
public class Dog extends Animal {
    String breed;

    public Dog(String name, String breed) {
        super(name);  // Call parent constructor
        this.breed = breed;
    }

    // New method specific to Dog
    public void fetch() {
        System.out.println(name + " is fetching the ball!");
    }
}

// Using inheritance
Dog dog = new Dog("Buddy", "Golden Retriever");
dog.eat();    // Buddy is eating.        (inherited)
dog.speak();  // Buddy makes a sound.    (inherited)
dog.fetch();  // Buddy is fetching the ball!  (own method)

Polymorphism

Polymorphism means "many forms" — it allows objects of different classes to be treated through a common interface. In Java, this is achieved through method overriding, where a child class provides its own implementation of a method defined in the parent class. Use the @Override annotation to signal that a method is being overridden.

java 
public class Animal {
    String name;

    public Animal(String name) {
        this.name = name;
    }

    public void speak() {
        System.out.println(name + " makes a sound.");
    }
}

public class Dog extends Animal {
    public Dog(String name) {
        super(name);
    }

    @Override
    public void speak() {
        System.out.println(name + " says: Woof!");
    }
}

public class Cat extends Animal {
    public Cat(String name) {
        super(name);
    }

    @Override
    public void speak() {
        System.out.println(name + " says: Meow!");
    }
}

// Polymorphism in action
Animal[] animals = {
    new Dog("Rex"),
    new Cat("Whiskers"),
    new Animal("Bird")
};

for (Animal a : animals) {
    a.speak();  // Each calls its own version
}
// Rex says: Woof!
// Whiskers says: Meow!
// Bird makes a sound.

Abstract Classes and Interfaces

An abstract class cannot be instantiated directly — it serves as a base class that defines a contract for its subclasses. An interface is a fully abstract type that specifies methods a class must implement. A class can extend one abstract class but implement multiple interfaces.

java 
// Abstract class
public abstract class Shape {
    String color;

    public Shape(String color) {
        this.color = color;
    }

    // Abstract method — must be implemented by subclasses
    public abstract double area();

    // Concrete method — shared by all subclasses
    public void describe() {
        System.out.println("A " + color + " shape with area " + area());
    }
}

// Interface
public interface Drawable {
    void draw();  // implicitly public and abstract
}

// Class that extends abstract class and implements interface
public class Circle extends Shape implements Drawable {
    private double radius;

    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }

    @Override
    public double area() {
        return Math.PI * radius * radius;
    }

    @Override
    public void draw() {
        System.out.println("Drawing a " + color + " circle with radius " + radius);
    }
}

// Using abstract class and interface
Circle c = new Circle("red", 5.0);
c.describe();  // A red shape with area 78.539...
c.draw();      // Drawing a red circle with radius 5.0
