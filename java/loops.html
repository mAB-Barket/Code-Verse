Loops in Java

Loops allow you to execute a block of code repeatedly based on a condition. Java provides several loop constructs—for, while, do-while, and the enhanced for (for-each)—each suited for different scenarios. Mastering loops is essential for iterating over data, automating repetitive tasks, and building efficient programs.

for Loop

The for loop is ideal when you know the exact number of iterations in advance. It combines initialization, condition checking, and iteration into a single, compact statement.

java 
// Print numbers 0 through 4
for (int i = 0; i < 5; i++) {
    System.out.println("i = " + i);
}

// Output:
// i = 0
// i = 1
// i = 2
// i = 3
// i = 4

The loop header contains three parts separated by semicolons: initialization (int i = 0), condition (i < 5), and update (i++). The loop body executes as long as the condition evaluates to true.

while Loop

The while loop repeats a block of code as long as a given condition remains true. It is best used when the number of iterations is not known ahead of time.

java 
int count = 0;

while (count < 5) {
    System.out.println("count = " + count);
    count++;
}

// Output:
// count = 0
// count = 1
// count = 2
// count = 3
// count = 4

The condition is evaluated before each iteration. If the condition is false from the start, the loop body never executes.

do-while Loop

The do-while loop is similar to the while loop, but the condition is checked after the loop body executes. This guarantees the body runs at least once, even if the condition is initially false.

java 
int num = 10;

do {
    System.out.println("num = " + num);
    num++;
} while (num < 5);

// Output:
// num = 10
// (executes once even though 10 < 5 is false)
💡

Tip: Use do-while when you need the loop body to execute at least once, for example when prompting the user for valid input before checking the condition.

Enhanced for Loop (for-each)

The enhanced for loop provides a cleaner syntax for iterating over arrays and collections. You don't need to manage an index variable—Java handles the traversal automatically.

java 
String[] fruits = {"Apple", "Banana", "Cherry", "Date"};

// Enhanced for loop (for-each)
for (String fruit : fruits) {
    System.out.println(fruit);
}

// Output:
// Apple
// Banana
// Cherry
// Date

// Works with any Iterable (ArrayList, HashSet, etc.)
import java.util.ArrayList;

ArrayList<Integer> numbers = new ArrayList<>();
numbers.add(10);
numbers.add(20);
numbers.add(30);

for (int n : numbers) {
    System.out.println(n);
}

break and continue

Java provides two keywords to alter the normal flow of loops. break immediately terminates the loop, while continue skips the rest of the current iteration and moves to the next one.

java 
// break — exit the loop when i equals 3
for (int i = 0; i < 10; i++) {
    if (i == 3) {
        break;
    }
    System.out.println(i);
}
// Output: 0  1  2

// continue — skip even numbers
for (int i = 0; i < 6; i++) {
    if (i % 2 == 0) {
        continue;
    }
    System.out.println(i);
}
// Output: 1  3  5
💡

Tip: Use break and continue sparingly. Overusing them can make your loop logic harder to follow. In most cases, a well-structured condition is clearer than a loop peppered with control-flow jumps.
