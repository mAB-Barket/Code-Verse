2. Variables in C++

📚 Lesson 2 of 7 ⏱ 12 min read 🌱 Beginner

Variables are fundamental building blocks in C++ programming that store data values which can be manipulated during program execution. Each variable must be declared with a specific data type that determines what kind of data it can hold.

// Basic variable declaration and initialization
int age = 25;
float price = 99.99;
char grade = 'A';
bool isActive = true;

Variables can have different scopes and lifetimes:

  • Local variables: Declared inside functions
  • Global variables: Declared outside functions
  • Static variables: Retain values between function calls
  • Constants: Values that cannot be changed
