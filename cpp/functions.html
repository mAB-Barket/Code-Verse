16. Function Basics

📚 Lesson 16 of 55 ⏱ 18 min read 🌱 Beginner

What Are Functions?

A function is a reusable block of code that performs a specific task. Functions help organize code into logical units, avoid repetition, and make programs easier to read, test, and maintain.

You have already been using functions: main() is a function, cout uses functions internally, and sqrt() from <cmath> is a function.

In this lesson you will learn:

  • Why functions are essential in programming
  • Function declaration (prototype) and definition
  • Calling functions
  • Function parameters and arguments
  • Return types and the return statement
  • void functions
  • Function scope and local variables
  • Forward declarations and prototypes

Key Point

Functions follow the DRY principle — Don’t Repeat Yourself. Instead of copying code, put it in a function and call it whenever needed.

Function Syntax

A C++ function has these parts:

returnType functionName(parameterList) {
    // function body
    return value;  // if returnType is not void
}
  • returnType — the data type of the value the function returns (int, double, string, void, etc.)
  • functionName — a descriptive name following C++ naming rules
  • parameterList — input values the function accepts (can be empty)
  • function body — the code that runs when the function is called
  • return — sends a value back to the caller

Your First Function

#include <iostream>
using namespace std;

// Function definition
void greet() {
    cout << "Hello from a function!" << endl;
}

int main() {
    greet();   // Function call
    greet();   // Call it again
    return 0;
}

Output:

Hello from a function!
Hello from a function!

Functions with Parameters

Parameters let you pass data into a function:

#include <iostream>
using namespace std;

void greetUser(string name) {
    cout << "Hello, " << name << "!" << endl;
}

int main() {
    greetUser("Alice");
    greetUser("Bob");
    return 0;
}

Output:

Hello, Alice!
Hello, Bob!

Multiple Parameters

#include <iostream>
using namespace std;

void addNumbers(int a, int b) {
    cout << a << " + " << b << " = " << a + b << endl;
}

int main() {
    addNumbers(5, 3);
    addNumbers(10, 20);
    return 0;
}

Tip

Parameters are the variables listed in the function definition. Arguments are the actual values passed when calling the function. People often use these terms interchangeably, but technically they are different.

Functions with Return Values

Functions can compute and return a value:

#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

double average(double x, double y) {
    return (x + y) / 2.0;
}

int main() {
    int sum = add(5, 3);
    cout << "Sum: " << sum << endl;

    double avg = average(10.5, 20.3);
    cout << "Average: " << avg << endl;

    // Use directly in expressions
    cout << "Direct: " << add(100, 200) << endl;
    return 0;
}

Output:

Sum: 8
Average: 15.4
Direct: 300

Key Point

The return statement immediately exits the function and sends the value back to the caller. Code after return will not execute.

void Functions

Functions that don’t return a value use the void return type:

#include <iostream>
using namespace std;

void printLine(int length) {
    for (int i = 0; i < length; i++) {
        cout << "-";
    }
    cout << endl;
}

void printBox(string text) {
    int len = text.length() + 4;
    printLine(len);
    cout << "| " << text << " |" << endl;
    printLine(len);
}

int main() {
    printBox("Hello");
    printBox("CodeVerse");
    return 0;
}

A void function can use return; (with no value) to exit early, but doesn’t need a return statement at the end.

Function Prototypes (Forward Declarations)

In C++, a function must be declared before it is called. If you define a function after main(), you need a prototype:

#include <iostream>
using namespace std;

// Function prototype (declaration)
int multiply(int a, int b);

int main() {
    cout << multiply(4, 5) << endl;  // Works because of prototype
    return 0;
}

// Function definition (after main)
int multiply(int a, int b) {
    return a * b;
}

The prototype tells the compiler: “This function exists, takes these parameters, and returns this type.”

Tip

Prototypes are essential when you have multiple functions that call each other, or when organizing code across multiple files. Parameter names in prototypes are optional: int multiply(int, int); is also valid.

Function Scope

Variables declared inside a function are local to that function. They cannot be accessed from outside:

#include <iostream>
using namespace std;

void myFunction() {
    int x = 10;  // Local variable
    cout << "Inside: " << x << endl;
}

int main() {
    myFunction();
    // cout << x;  // ERROR: x is not accessible here
    return 0;
}

Key Point

Each function call creates its own set of local variables. When the function ends, those variables are destroyed. This is called automatic storage duration.

The main() Function

The main() function is special in C++:

  • It is the entry point of every C++ program.
  • It returns int — conventionally 0 for success, non-zero for failure.
  • It can accept command-line arguments: int main(int argc, char* argv[])
  • You cannot call main() from your own code (it’s called by the runtime system).

Practical Examples

Example 1: Temperature Converter

#include <iostream>
using namespace std;

double celsiusToFahrenheit(double celsius) {
    return (celsius * 9.0 / 5.0) + 32.0;
}

double fahrenheitToCelsius(double fahrenheit) {
    return (fahrenheit - 32.0) * 5.0 / 9.0;
}

int main() {
    cout << "100 C = " << celsiusToFahrenheit(100) << " F" << endl;
    cout << "212 F = " << fahrenheitToCelsius(212) << " C" << endl;
    return 0;
}

Example 2: Math Utilities

#include <iostream>
using namespace std;

int maximum(int a, int b) {
    return (a > b) ? a : b;
}

int minimum(int a, int b) {
    return (a < b) ? a : b;
}

bool isEven(int n) {
    return n % 2 == 0;
}

bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}

int main() {
    cout << "Max(5,3): " << maximum(5, 3) << endl;
    cout << "Min(5,3): " << minimum(5, 3) << endl;
    cout << "isEven(4): " << isEven(4) << endl;
    cout << "isPrime(17): " << isPrime(17) << endl;
    return 0;
}

Example 3: Functions Calling Functions

#include <iostream>
using namespace std;

int square(int n) {
    return n * n;
}

int sumOfSquares(int a, int b) {
    return square(a) + square(b);  // Calling square() inside
}

int main() {
    cout << "3^2 + 4^2 = " << sumOfSquares(3, 4) << endl;  // 25
    return 0;
}

Summary

  • A function is a reusable block of code with a name, parameters, and return type.
  • void functions perform actions without returning a value.
  • Non-void functions use return to send a value back to the caller.
  • Parameters are defined in the function; arguments are passed when calling it.
  • Variables inside functions are local — they exist only during the function call.
  • Function prototypes let you declare a function before defining it.
  • Functions can call other functions, including themselves (recursion).
  • Well-named functions make code self-documenting.
