13. While & Do-While Loops

📚 Lesson 13 of 55 ⏱ 20 min read 🌱 Beginner

What Is a While Loop?

In the previous lesson you learned the for loop, which is ideal when you know how many times to repeat. But what if you don’t know in advance? That’s where the while loop shines.

A while loop repeats a block of code as long as a condition is true. The number of iterations is determined at runtime, making it perfect for situations where you must keep going until something specific happens.

In this lesson you will learn:

  • The syntax and execution flow of a while loop
  • How while differs from for
  • The do-while loop and when to use it
  • Infinite loops with while(true)
  • Sentinel-controlled loops
  • Counter-controlled while loops
  • Input validation with loops
  • Common mistakes: infinite loops, off-by-one errors
  • Practical examples: guessing game, menu loop, digit counter, reverse number, GCD calculation

Key Point

Use a while loop when the number of iterations depends on a condition evaluated at runtime. Use a for loop when you know in advance how many times to repeat.

While Loop Syntax

The general form of a while loop is:

while (condition) {
    // Code to repeat
}

How it works:

  • Condition — evaluated before each iteration. If true, the loop body runs; if false, the loop ends.
  • The condition must eventually become false, or you get an infinite loop.
  • If the condition is false from the start, the body never executes.

Tip

Always make sure something inside the loop body changes the condition so the loop can eventually end. Forgetting to update the condition variable is the most common cause of infinite loops.

How a While Loop Executes

Here is the step-by-step execution order:

  1. Condition is evaluated. If false, the loop exits immediately.
  2. The loop body executes.
  3. Go back to step 1.
// Trace example
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 3) {
        cout << "i = " << i << endl;
        i++;
    }
    return 0;
}

// Step-by-step:
// 1. i = 1, 1 <= 3? true  → print "i = 1" → i++ → i = 2
// 2. i = 2, 2 <= 3? true  → print "i = 2" → i++ → i = 3
// 3. i = 3, 3 <= 3? true  → print "i = 3" → i++ → i = 4
// 4. i = 4, 4 <= 3? false → loop ends

Output:

i = 1
i = 2
i = 3

Key Point

The condition is checked before the body runs. If it is false from the start, the body executes zero times. This is the key difference from a do-while loop.

Counter-Controlled While Loops

You can use a while loop as a counter, just like a for loop. You just need to manage the counter variable yourself.

#include <iostream>
using namespace std;

int main() {
    // Print numbers 1 through 5
    int i = 1;          // initialization
    while (i <= 5) {   // condition
        cout << i << " ";
        i++;            // update
    }
    cout << endl;
    return 0;
}

Output:

1 2 3 4 5

Counting Down

#include <iostream>
using namespace std;

int main() {
    int count = 5;
    while (count >= 1) {
        cout << count << " ";
        count--;
    }
    cout << endl;
    return 0;
}

Output:

5 4 3 2 1

Tip

When you know the exact number of iterations, a for loop is usually cleaner. Use while when the loop’s continuation depends on a runtime condition rather than a simple counter.

Sentinel-Controlled Loops

A sentinel is a special value that signals the end of input. The loop continues until the sentinel value is encountered.

#include <iostream>
using namespace std;

int main() {
    int number;
    int sum = 0;

    cout << "Enter numbers (enter -1 to stop):" << endl;
    cin >> number;

    while (number != -1) {
        sum += number;
        cin >> number;
    }

    cout << "Total sum: " << sum << endl;
    return 0;
}

Sample interaction:

Enter numbers (enter -1 to stop):
10
20
30
-1
Total sum: 60

Key Point

In a sentinel-controlled loop, the first read happens before the loop (called a priming read). Inside the loop body, you read the next value at the end so the condition is checked with fresh data on the next iteration.

Do-While Loop Syntax

The do-while loop is similar to the while loop, but with one critical difference: the condition is checked after the body executes, guaranteeing at least one execution.

do {
    // Code to repeat
} while (condition);

Important: Notice the semicolon after the closing parenthesis. This is required and easy to forget!

How It Executes

  1. The loop body executes.
  2. The condition is evaluated. If true, go back to step 1; if false, the loop ends.
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    do {
        cout << "i = " << i << endl;
        i++;
    } while (i <= 3);
    return 0;
}

// Step-by-step:
// 1. print "i = 1" → i++ → i = 2 → 2 <= 3? true
// 2. print "i = 2" → i++ → i = 3 → 3 <= 3? true
// 3. print "i = 3" → i++ → i = 4 → 4 <= 3? false → loop ends

Output:

i = 1
i = 2
i = 3

While vs Do-While

The key difference is when the condition is checked:

// while: checks BEFORE the body
int x = 10;
while (x < 5) {
    cout << x << " ";   // Never executes!
    x++;
}

// do-while: checks AFTER the body
int y = 10;
do {
    cout << y << " ";   // Executes once! Prints: 10
    y++;
} while (y < 5);

Key Point

while may execute its body zero or more times. do-while always executes its body at least once. Use do-while when you need something to happen before checking the condition — such as displaying a menu or getting user input.

Input Validation with Do-While

One of the most common uses for do-while is input validation: ask the user for input, then check if it’s valid. If not, ask again.

#include <iostream>
using namespace std;

int main() {
    int age;

    do {
        cout << "Enter your age (1-120): ";
        cin >> age;
        if (age < 1 || age > 120) {
            cout << "Invalid age. Try again." << endl;
        }
    } while (age < 1 || age > 120);

    cout << "Your age: " << age << endl;
    return 0;
}

Sample interaction:

Enter your age (1-120): -5
Invalid age. Try again.
Enter your age (1-120): 200
Invalid age. Try again.
Enter your age (1-120): 25
Your age: 25

Tip

do-while is perfect for input validation because you always need to ask the user at least once before you can check whether the input is valid.

Infinite Loops with while(true)

Sometimes you want a loop that runs forever (or until you explicitly break out). Use while(true) for this pattern.

#include <iostream>
using namespace std;

int main() {
    int num;

    while (true) {
        cout << "Enter a positive number (0 to quit): ";
        cin >> num;

        if (num == 0) {
            cout << "Goodbye!" << endl;
            break;   // Exit the loop
        }

        if (num < 0) {
            cout << "Negative numbers not allowed. Try again." << endl;
            continue; // Skip to next iteration
        }

        cout << "You entered: " << num << endl;
    }
    return 0;
}

Key Point

while(true) creates an intentional infinite loop. You must have a break statement (or a return) inside the loop to exit. Without one, the program will run forever.

While vs For Comparison

Both while and for can do the same things, but each is better suited for certain situations:

// FOR loop: best when count is known
for (int i = 0; i < 10; i++) {
    cout << i << " ";
}

// Equivalent WHILE loop
int i = 0;
while (i < 10) {
    cout << i << " ";
    i++;
}

When to use each:

  • for — when you know the iteration count beforehand (counting, array traversal).
  • while — when the loop depends on a condition that changes unpredictably (user input, file reading, searching).
  • do-while — when you need at least one execution (menus, input validation).

Common Mistakes

Mistake 1: Infinite Loop (Forgetting to Update)

// BUG: i is never updated, loop runs forever!
int i = 1;
while (i <= 5) {
    cout << i << " ";
    // Missing: i++;
}

Mistake 2: Off-by-One Error

// Intended: print 1-5, but prints 1-4
int i = 1;
while (i < 5) {   // Should be i <= 5
    cout << i << " ";
    i++;
}
// Output: 1 2 3 4  (missing 5!)

Mistake 3: Semicolon After while

// BUG: Semicolon makes the body empty!
int i = 1;
while (i <= 5);   // ← empty body, infinite loop!
{
    cout << i << " ";
    i++;
}

Mistake 4: Forgetting Semicolon in do-while

// BUG: Missing semicolon after do-while
int x = 1;
do {
    cout << x;
    x++;
} while (x <= 5)   // ← Missing semicolon! Compile error.

Tip

If your program seems to “freeze” or hang, you likely have an infinite loop. Check that your loop condition can become false, and make sure the variable in the condition is being updated inside the loop body.

Practical Examples

Example 1: Number Guessing Game

#include <iostream>
#include <cstdlib>
#include <ctime>
using namespace std;

int main() {
    srand(time(0));
    int secret = rand() % 100 + 1;  // 1-100
    int guess;
    int attempts = 0;

    cout << "Guess the number (1-100)!" << endl;

    do {
        cout << "Your guess: ";
        cin >> guess;
        attempts++;

        if (guess < secret) {
            cout << "Too low!" << endl;
        } else if (guess > secret) {
            cout << "Too high!" << endl;
        }
    } while (guess != secret);

    cout << "Correct! You got it in " << attempts << " attempts." << endl;
    return 0;
}

Example 2: Menu Loop

#include <iostream>
using namespace std;

int main() {
    int choice;

    do {
        cout << "\n===== MENU =====" << endl;
        cout << "1. Say Hello" << endl;
        cout << "2. Say Goodbye" << endl;
        cout << "3. Exit" << endl;
        cout << "Choice: ";
        cin >> choice;

        switch (choice) {
            case 1: cout << "Hello!" << endl; break;
            case 2: cout << "Goodbye!" << endl; break;
            case 3: cout << "Exiting..." << endl; break;
            default: cout << "Invalid choice." << endl;
        }
    } while (choice != 3);

    return 0;
}

Example 3: Count Digits of a Number

#include <iostream>
using namespace std;

int main() {
    int num;
    cout << "Enter a positive integer: ";
    cin >> num;

    int count = 0;
    int temp = num;

    while (temp > 0) {
        temp /= 10;   // Remove last digit
        count++;
    }

    // Special case: 0 has 1 digit
    if (num == 0) count = 1;

    cout << num << " has " << count << " digit(s)." << endl;
    return 0;
}

Output:

Enter a positive integer: 12345
12345 has 5 digit(s).

Example 4: Reverse a Number

#include <iostream>
using namespace std;

int main() {
    int num, reversed = 0;
    cout << "Enter a number: ";
    cin >> num;

    while (num > 0) {
        int digit = num % 10;      // Get last digit
        reversed = reversed * 10 + digit;  // Append digit
        num /= 10;                 // Remove last digit
    }

    cout << "Reversed: " << reversed << endl;
    return 0;
}

Output:

Enter a number: 1234
Reversed: 4321

Example 5: GCD Calculation (Euclidean Algorithm)

#include <iostream>
using namespace std;

int main() {
    int a, b;
    cout << "Enter two numbers: ";
    cin >> a >> b;

    // Euclidean algorithm
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }

    cout << "GCD: " << a << endl;
    return 0;
}

Output:

Enter two numbers: 48 18
GCD: 6

Example 6: Sum of Digits

#include <iostream>
using namespace std;

int main() {
    int num, sum = 0;
    cout << "Enter a number: ";
    cin >> num;

    while (num > 0) {
        sum += num % 10;   // Add last digit
        num /= 10;         // Remove last digit
    }

    cout << "Sum of digits: " << sum << endl;
    return 0;
}

Output:

Enter a number: 9876
Sum of digits: 30

Example 7: Power Calculation

#include <iostream>
using namespace std;

int main() {
    int base, exp;
    long long result = 1;

    cout << "Enter base and exponent: ";
    cin >> base >> exp;

    int i = 0;
    while (i < exp) {
        result *= base;
        i++;
    }

    cout << base << "^" << exp << " = " << result << endl;
    return 0;
}

Output:

Enter base and exponent: 2 10
2^10 = 1024

Summary

  • The while loop checks its condition before the body — the body may execute zero times.
  • The do-while loop checks its condition after the body — the body always executes at least once.
  • do-while requires a semicolon after the closing while(condition);.
  • Use while for sentinel-controlled loops, reading input, and searching.
  • Use do-while for menus, input validation, and any situation requiring at least one execution.
  • Use while(true) with break for deliberately infinite loops with explicit exit points.
  • Always update the loop variable inside the body to avoid infinite loops.
  • Beware of off-by-one errors: double-check < vs <= and initial/final values.
  • Common patterns include: digit extraction (num % 10, num /= 10), GCD, and input validation.
  • A while loop is equivalent to a for loop with the init and update moved outside the header.
