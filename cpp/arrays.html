22. Arrays

📚 Lesson 22 of 55 ⏱ 15 min read 🌱 Beginner

What Are Arrays?

An array is a fixed-size, contiguous collection of elements that all share the same data type. Instead of declaring dozens of individual variables, you can store related values in a single array and access each one through its numeric index.

In this lesson you will learn:

  • How to declare and initialize arrays
  • Accessing and modifying elements using zero-based indices
  • Determining array size with the sizeof operator
  • Looping through arrays with classic for and range-based for
  • Passing arrays to functions
  • Common operations — finding minimum, maximum, sum, and average
  • Array bounds and undefined behavior
  • The modern std::array alternative from C++11

Think of an array as a row of numbered mailboxes in a post office. Every mailbox is the same size (same type), they sit side by side in memory (contiguous), and you look up a specific mailbox by its number (index).

Declaring and Initializing Arrays

The general syntax for declaring a C-style array is:

type arrayName[size];

Here type is the data type of each element, arrayName is the identifier, and size is a compile-time constant specifying how many elements the array can hold.

Declaration Without Initialization

#include <iostream>
using namespace std;

int main() {
    int scores[5];  // declares an array of 5 ints (values are indeterminate)
    return 0;
}

When an array is declared without an initializer inside a function, its elements contain indeterminate (garbage) values. Always initialize your arrays before reading from them.

Initialization with an Initializer List

#include <iostream>
using namespace std;

int main() {
    // Full initialization
    int scores[5] = {90, 85, 78, 92, 88};

    // Partial initialization — remaining elements are zero
    int data[5] = {10, 20};  // data = {10, 20, 0, 0, 0}

    // Zero-initialize all elements
    int zeros[5] = {};        // zeros = {0, 0, 0, 0, 0}

    // Let the compiler deduce the size from the initializer
    int nums[] = {1, 2, 3, 4};  // size is automatically 4

    return 0;
}

Uniform Initialization (C++11)

int values[3]{10, 20, 30};       // no = sign needed
double temps[4]{36.5, 37.0, 36.8, 37.2};

Accessing Elements with Index

Array indices in C++ are zero-based — the first element is at index 0, the second at index 1, and the last element of an array of size n is at index n−1.

#include <iostream>
using namespace std;

int main() {
    int scores[5] = {90, 85, 78, 92, 88};

    cout << "First element : " << scores[0] << endl;  // 90
    cout << "Third element : " << scores[2] << endl;  // 78
    cout << "Last element  : " << scores[4] << endl;  // 88

    return 0;
}

Output:

First element : 90
Third element : 78
Last element  : 88

Modifying Array Elements

You can change any element by assigning a new value through its index:

#include <iostream>
using namespace std;

int main() {
    int scores[5] = {90, 85, 78, 92, 88};

    scores[1] = 95;   // change second element from 85 to 95
    scores[4] = 100;  // change last element from 88 to 100

    // Print all elements
    for (int i = 0; i < 5; i++) {
        cout << "scores[" << i << "] = " << scores[i] << endl;
    }

    return 0;
}

Output:

scores[0] = 90
scores[1] = 95
scores[2] = 78
scores[3] = 92
scores[4] = 100

Array Size and the sizeof Operator

The sizeof operator returns the total number of bytes occupied by the array. To get the number of elements, divide by the size of one element:

#include <iostream>
using namespace std;

int main() {
    int nums[] = {10, 20, 30, 40, 50};

    int totalBytes   = sizeof(nums);        // e.g. 20 (5 × 4 bytes)
    int elementBytes = sizeof(nums[0]);     // e.g. 4
    int length       = sizeof(nums) / sizeof(nums[0]);  // 5

    cout << "Total bytes   : " << totalBytes   << endl;
    cout << "Element bytes : " << elementBytes << endl;
    cout << "Array length  : " << length       << endl;

    return 0;
}

Output (typical 64-bit system):

Total bytes   : 20
Element bytes : 4
Array length  : 5

Alternatively, since C++17 you can use std::size() from <iterator>:

#include <iostream>
#include <iterator>
using namespace std;

int main() {
    int nums[] = {10, 20, 30, 40, 50};
    cout << "Length: " << size(nums) << endl;  // 5
    return 0;
}

Looping Through Arrays

Classic for Loop

#include <iostream>
using namespace std;

int main() {
    string fruits[] = {"Apple", "Banana", "Cherry", "Date", "Elderberry"};
    int len = sizeof(fruits) / sizeof(fruits[0]);

    for (int i = 0; i < len; i++) {
        cout << i << ": " << fruits[i] << endl;
    }

    return 0;
}

Output:

0: Apple
1: Banana
2: Cherry
3: Date
4: Elderberry

Range-Based for Loop (C++11)

The range-based for loop iterates over every element without needing an index variable:

#include <iostream>
using namespace std;

int main() {
    int scores[] = {90, 85, 78, 92, 88};

    for (int s : scores) {
        cout << s << " ";
    }
    cout << endl;

    return 0;
}
// Output: 90 85 78 92 88

Use a reference (&) if you need to modify elements in place:

for (int& s : scores) {
    s += 5;  // adds 5 to every score
}

Use const auto& for read-only access without copying:

for (const auto& s : scores) {
    cout << s << " ";
}

Arrays and Functions

When you pass a C-style array to a function, it decays into a pointer to its first element. This means the function does not know the array’s size — you must pass the size separately.

Passing an Array to a Function

#include <iostream>
using namespace std;

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    int nums[] = {10, 20, 30, 40, 50};
    int len = sizeof(nums) / sizeof(nums[0]);

    printArray(nums, len);  // Output: 10 20 30 40 50

    return 0;
}

Because the array decays to a pointer, any changes the function makes to the elements affect the original array (it is not a copy).

Preventing Modification with const

void printArray(const int arr[], int size) {
    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    // arr[0] = 999;  // ERROR — cannot modify a const array
}

Returning an Array from a Function

You cannot return a C-style array directly from a function. Common alternatives include:

  • Pass the destination array as a parameter and fill it inside the function.
  • Use std::array or std::vector, which can be returned.
  • Allocate a dynamic array with new[] and return the pointer (but then you must remember to delete[]).

Common Array Operations

Finding the Minimum and Maximum

#include <iostream>
using namespace std;

int main() {
    int nums[] = {34, 12, 56, 7, 89, 23};
    int len = sizeof(nums) / sizeof(nums[0]);

    int minVal = nums[0];
    int maxVal = nums[0];

    for (int i = 1; i < len; i++) {
        if (nums[i] < minVal) minVal = nums[i];
        if (nums[i] > maxVal) maxVal = nums[i];
    }

    cout << "Min: " << minVal << endl;  // 7
    cout << "Max: " << maxVal << endl;  // 89

    return 0;
}

Calculating Sum and Average

#include <iostream>
using namespace std;

int main() {
    double grades[] = {88.5, 92.0, 76.5, 95.0, 81.0};
    int len = sizeof(grades) / sizeof(grades[0]);

    double sum = 0;
    for (int i = 0; i < len; i++) {
        sum += grades[i];
    }

    double average = sum / len;

    cout << "Sum:     " << sum     << endl;  // 433
    cout << "Average: " << average << endl;  // 86.6

    return 0;
}

Reversing an Array

#include <iostream>
using namespace std;

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int len = sizeof(arr) / sizeof(arr[0]);

    // Swap elements from both ends toward the center
    for (int i = 0; i < len / 2; i++) {
        int temp = arr[i];
        arr[i] = arr[len - 1 - i];
        arr[len - 1 - i] = temp;
    }

    for (int x : arr) cout << x << " ";
    cout << endl;
    // Output: 5 4 3 2 1

    return 0;
}

Linear Search

#include <iostream>
using namespace std;

int linearSearch(const int arr[], int size, int target) {
    for (int i = 0; i < size; i++) {
        if (arr[i] == target) return i;  // found at index i
    }
    return -1;  // not found
}

int main() {
    int data[] = {5, 12, 8, 3, 17, 9};
    int len = sizeof(data) / sizeof(data[0]);

    int idx = linearSearch(data, len, 17);
    if (idx != -1)
        cout << "Found 17 at index " << idx << endl;
    else
        cout << "17 not found" << endl;

    return 0;
}
// Output: Found 17 at index 4

Array Bounds and Undefined Behavior

C++ does not perform automatic bounds checking on C-style arrays. Accessing an index that is outside the valid range (0 to size−1) results in undefined behavior — your program might crash, produce garbage values, or silently corrupt memory.

#include <iostream>
using namespace std;

int main() {
    int arr[3] = {10, 20, 30};

    // DANGEROUS — undefined behavior!
    // cout << arr[5]  << endl;   // reading out of bounds
    // arr[-1] = 99;               // writing out of bounds

    // SAFE — always validate indices
    int index = 2;
    if (index >= 0 && index < 3) {
        cout << arr[index] << endl;  // 30
    }

    return 0;
}

Best practice: Always ensure your loop counters and indices stay within the valid range. Use std::array with .at() if you want runtime bounds checking.

std::array — The Modern Alternative (C++11)

The <array> header provides std::array, a fixed-size container that wraps a C-style array while adding safety and convenience features:

  • Knows its own size (.size())
  • Supports bounds-checked access via .at()
  • Works seamlessly with STL algorithms and range-based for
  • Can be returned from functions and compared with ==
  • Does not decay to a pointer when passed to functions

Declaring and Using std::array

#include <iostream>
#include <array>
using namespace std;

int main() {
    // Declare and initialize
    array<int, 5> scores = {90, 85, 78, 92, 88};

    // Size is always known
    cout << "Size: " << scores.size() << endl;  // 5

    // Access elements
    cout << "First: " << scores[0]    << endl;  // 90
    cout << "Last:  " << scores.back() << endl;  // 88

    // Bounds-checked access
    try {
        cout << scores.at(10) << endl;  // throws std::out_of_range
    } catch (const out_of_range& e) {
        cout << "Error: " << e.what() << endl;
    }

    // Range-based for loop
    for (int s : scores) {
        cout << s << " ";
    }
    cout << endl;

    return 0;
}

Passing std::array to Functions

#include <iostream>
#include <array>
using namespace std;

void printScores(const array<int, 5>& arr) {
    for (int s : arr) {
        cout << s << " ";
    }
    cout << endl;
}

array<int, 5> doubleScores(const array<int, 5>& arr) {
    array<int, 5> result;
    for (size_t i = 0; i < arr.size(); i++) {
        result[i] = arr[i] * 2;
    }
    return result;  // std::array CAN be returned!
}

int main() {
    array<int, 5> scores = {10, 20, 30, 40, 50};
    printScores(scores);

    auto doubled = doubleScores(scores);
    printScores(doubled);  // 20 40 60 80 100

    return 0;
}

Useful std::array Member Functions

#include <iostream>
#include <array>
#include <algorithm>
using namespace std;

int main() {
    array<int, 5> a = {50, 10, 40, 20, 30};

    // front() and back()
    cout << "Front: " << a.front() << endl;  // 50
    cout << "Back:  " << a.back()  << endl;  // 30

    // empty() and size()
    cout << "Empty? " << a.empty() << endl;  // 0 (false)
    cout << "Size:  " << a.size()  << endl;  // 5

    // fill() — set all elements to a value
    a.fill(0);
    // a = {0, 0, 0, 0, 0}

    // sort with STL algorithm
    array<int, 5> b = {50, 10, 40, 20, 30};
    sort(b.begin(), b.end());
    for (int x : b) cout << x << " ";  // 10 20 30 40 50
    cout << endl;

    return 0;
}

Comparison: C-Style Array vs std::array

Feature C-Style Array std::array
Knows its size No (decays to pointer) Yes (.size())
Bounds checking No Yes (.at())
Returnable from function No Yes
Assignable / Copyable No Yes
Works with STL algorithms Via pointers Via .begin() / .end()
Comparison (==, <) No (compares addresses) Yes (element-wise)
Performance Zero overhead Zero overhead (thin wrapper)
Header needed None <array>

Recommendation: Prefer std::array over C-style arrays in modern C++ code. It provides the same performance with added safety and convenience.

Key Takeaways

  • An array is a fixed-size, contiguous collection of elements of the same type.
  • Declare arrays with type name[size]; and initialize with curly braces: {val1, val2, ...}.
  • Array indices are zero-based — valid indices run from 0 to size − 1.
  • Use sizeof(arr) / sizeof(arr[0]) to compute the number of elements in a C-style array.
  • Use a classic for loop when you need the index, or a range-based for loop (C++11) for cleaner iteration.
  • Arrays decay to pointers when passed to functions — always pass the size as a separate argument.
  • Accessing out-of-bounds indices is undefined behavior — C++ does not check automatically.
  • std::array (C++11) is a modern, zero-overhead wrapper that knows its size, supports bounds checking via .at(), and works seamlessly with STL algorithms.
  • Prefer std::array over C-style arrays for new code — or use std::vector when you need a dynamic (resizable) array.
  • Common operations like finding min/max, sum, average, search, and reversal are fundamental patterns for working with arrays.
