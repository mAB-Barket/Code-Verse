25. std::string

📚 Lesson 25 of 55 ⏱ 16 min read 🔨 Intermediate

What is std::string?

std::string is a dynamic string class provided by the C++ Standard Library through the <string> header. Unlike C-strings (null-terminated char arrays), std::string manages its own memory automatically — it grows and shrinks as needed without you worrying about buffer sizes or null terminators.

Under the hood, std::string is a typedef for std::basic_string<char>. It stores characters contiguously in heap memory and tracks its own length, making operations like querying the size an O(1) operation.

In this lesson you will learn:

  • How to create, initialize, and assign strings
  • Reading strings from the keyboard with cin and getline
  • Querying length, capacity, and emptiness
  • Accessing individual characters safely
  • Concatenating and comparing strings
  • Searching, extracting substrings, and modifying content
  • Iterating over strings with range-based for loops and iterators
  • Converting between strings and numbers
  • A handy reference table of the most useful methods

Creating and Initializing Strings

The <string> header provides many constructors for std::string:

Common Initialization Methods

#include <string>
using namespace std;

string s1;                     // Empty string ""
string s2 = "Hello";           // From a string literal
string s3("World");            // Direct initialization
string s4(5, 'A');             // "AAAAA" — 5 copies of 'A'
string s5(s2);                 // Copy of s2: "Hello"
string s6(s2, 1, 3);          // Substring of s2 starting at index 1, length 3: "ell"
string s7 = s2 + " " + s3;   // Concatenation: "Hello World"

Assignment After Declaration

string name;
name = "CodeVerse";            // Assign a literal
name = s2;                     // Assign another string
name.assign("C++");            // Using the assign() method
name.assign(10, '-');          // "----------" — 10 dashes

Unlike C-strings, you never need to worry about buffer sizes — std::string allocates and reallocates memory as needed.

String Input: cin and getline

Using cin >>

#include <iostream>
#include <string>
using namespace std;

int main() {
    string firstName;
    cout << "Enter your first name: ";
    cin >> firstName;           // Reads one word (stops at whitespace)
    cout << "Hello, " << firstName << "!\n";
    return 0;
}

cin >> reads a single word — it stops at the first whitespace character. For full lines, use getline.

Using getline

#include <iostream>
#include <string>
using namespace std;

int main() {
    string fullName;
    cout << "Enter your full name: ";
    getline(cin, fullName);     // Reads the entire line (including spaces)
    cout << "Hello, " << fullName << "!\n";
    return 0;
}

getline(cin, str) reads everything up to and including the newline character, discards the newline, and stores the rest in str.

Mixing cin >> and getline

int age;
string city;

cout << "Age: ";
cin >> age;
cin.ignore();                  // Discard the leftover newline
cout << "City: ";
getline(cin, city);            // Now works correctly

After cin >>, a leftover '\n' remains in the input buffer. Call cin.ignore() before getline to consume it; otherwise getline reads an empty string.

String Length and Capacity

string s = "Hello, World!";

cout << s.size();       // 13 — number of characters
cout << s.length();     // 13 — identical to size()
cout << s.capacity();   // ≥ 13 — allocated storage (implementation-defined)
cout << s.empty();      // false (0) — string is not empty
cout << s.max_size();   // Maximum possible string size (very large)

string blank;
cout << blank.empty();  // true (1)

size() and length() are identical — both return the number of characters in O(1) time. capacity() returns how much memory is currently allocated; it is always ≥ size(). empty() returns true if the string has zero characters.

Resizing and Reserving

string s = "Hi";
s.reserve(100);       // Pre-allocate space for 100 chars (avoids reallocations)
cout << s.capacity();  // ≥ 100

s.resize(10, '.');    // s = "Hi........" (pads with '.')
s.resize(2);          // s = "Hi" (truncates)
s.shrink_to_fit();    // Request to reduce capacity to match size

Accessing Characters

string s = "CodeVerse";

// Using [] — no bounds checking (faster, undefined behavior if out of range)
char first = s[0];       // 'C'
char last  = s[8];       // 'e'
s[0] = 'c';              // s = "codeVerse"

// Using at() — bounds-checked (throws std::out_of_range if invalid)
char ch = s.at(4);       // 'V'
// s.at(100);            // Throws std::out_of_range exception

// front() and back()
char f = s.front();      // First character: 'c'
char b = s.back();       // Last character: 'e'

Use [] when you are certain the index is valid for maximum performance. Use at() when safety is more important — it throws an exception on invalid access instead of causing undefined behavior.

String Concatenation

string a = "Hello";
string b = " World";

// Using the + operator
string c = a + b;            // "Hello World"
string d = a + "!!!";        // "Hello!!!"

// Using += operator
a += b;                      // a = "Hello World"
a += '!';                    // a = "Hello World!"

// Using append()
string e = "Code";
e.append("Verse");           // e = "CodeVerse"
e.append(3, '!');            // e = "CodeVerse!!!"
e.append(b, 0, 6);          // Append substring of b: e = "CodeVerse!!! World"

The + operator creates a new string. For building strings in a loop, prefer += or append() to avoid unnecessary copies.

Important: You Cannot Concatenate Two Literals

// string s = "Hello" + " World";  // ERROR: can't add two const char*
string s = string("Hello") + " World";   // OK
string t = "Hello"s + " World";          // OK (C++14 string literal suffix)

String Comparison

string a = "apple";
string b = "banana";
string c = "apple";

// Relational operators (lexicographic comparison)
cout << (a == c);   // true  (1)
cout << (a != b);   // true  (1)
cout << (a < b);    // true  (1) — "apple" comes before "banana"
cout << (b > a);    // true  (1)
cout << (a <= c);   // true  (1)
cout << (a >= c);   // true  (1)

// Using compare()
int result = a.compare(b);
// result < 0: a is less than b
// result == 0: a equals b
// result > 0: a is greater than b

// Comparing substrings
a.compare(0, 3, b, 0, 3);   // Compare "app" vs "ban"

Unlike C-strings, you can use ==, !=, <, > directly with std::string. These perform proper lexicographic comparison of the string contents, not pointer addresses.

Searching Strings

string s = "Learn C++ at CodeVerse";

// find() — returns index of first occurrence, or string::npos if not found
size_t pos = s.find("C++");         // 6
size_t bad = s.find("Python");      // string::npos

if (s.find("Code") != string::npos)
    cout << "Found 'Code'!\n";

// find() with starting position
size_t pos2 = s.find('e', 2);       // 5 — first 'e' at or after index 2

// rfind() — searches backwards (last occurrence)
size_t rpos = s.rfind('e');          // 21 — last 'e' in the string

// find_first_of() — first occurrence of ANY character in the set
size_t vowel = s.find_first_of("aeiou");   // 1 ('e' in "Learn")

// find_last_of() — last occurrence of ANY character in the set
size_t lastVowel = s.find_last_of("aeiou");  // 20 ('e' in "Verse")

// find_first_not_of() — first char NOT in the set
size_t notSpace = s.find_first_not_of(" ");  // 0 ('L')

Understanding string::npos

// string::npos is a special constant (typically the maximum value of size_t)
// It means "not found" or "until the end of the string"
if (s.find("xyz") == string::npos) {
    cout << "Substring not found\n";
}

Substring, Replace, Erase, and Insert

substr() — Extract a Substring

string s = "Hello, World!";
string sub1 = s.substr(7);       // "World!" — from index 7 to end
string sub2 = s.substr(0, 5);    // "Hello" — from index 0, length 5

replace() — Replace Part of a String

string s = "Hello, World!";
s.replace(7, 5, "C++");          // s = "Hello, C++!"
// Replaces 5 characters starting at index 7 with "C++"

erase() — Remove Characters

string s = "Hello, World!";
s.erase(5, 7);                   // s = "Hello!" — erases 7 chars from index 5
s.erase(5);                      // s = "Hello" — erases from index 5 to end
s.erase();                       // s = "" — clears the string

insert() — Insert Characters

string s = "Hello!";
s.insert(5, ", World");          // s = "Hello, World!"
s.insert(0, ">> ");              // s = ">> Hello, World!"
s.insert(s.size(), " <<");       // s = ">> Hello, World! <<"

Combining Operations: Find and Replace All

string text = "I like cats. cats are great. I have 2 cats.";
string target = "cats";
string replacement = "dogs";

size_t pos = 0;
while ((pos = text.find(target, pos)) != string::npos) {
    text.replace(pos, target.length(), replacement);
    pos += replacement.length();  // Move past the replacement
}
// text = "I like dogs. dogs are great. I have 2 dogs."

String Iteration

Range-Based for Loop (C++11)

string s = "CodeVerse";

// Read-only
for (char ch : s) {
    cout << ch << ' ';
}
// Output: C o d e V e r s e

// Modify in place (use reference)
for (char& ch : s) {
    ch = toupper(ch);
}
// s = "CODEVERSE"

Index-Based Loop

for (size_t i = 0; i < s.size(); i++) {
    cout << s[i];
}

Using Iterators

// Forward iteration
for (string::iterator it = s.begin(); it != s.end(); ++it) {
    cout << *it;
}

// Using auto (C++11)
for (auto it = s.begin(); it != s.end(); ++it) {
    cout << *it;
}

// Reverse iteration
for (auto rit = s.rbegin(); rit != s.rend(); ++rit) {
    cout << *rit;
}
// Prints the string backwards

String Conversions

String to Number

#include <string>
using namespace std;

string numStr = "42";
string piStr  = "3.14159";
string fStr   = "2.5f";

int    n = stoi(numStr);      // 42
long   l = stol(numStr);      // 42L
double d = stod(piStr);       // 3.14159
float  f = stof(fStr);        // 2.5f
long long ll = stoll("123456789012345");  // 123456789012345LL

// stoi can handle different bases
int hex = stoi("FF", nullptr, 16);   // 255
int bin = stoi("1010", nullptr, 2);  // 10

If the string cannot be converted, these functions throw std::invalid_argument. If the value is out of range, they throw std::out_of_range.

Number to String

int age = 25;
double pi = 3.14159;

string s1 = to_string(age);     // "25"
string s2 = to_string(pi);      // "3.141590" (6 decimal places by default)

Converting to a C-String

string s = "Hello";
const char* cstr = s.c_str();   // Returns a const char* (null-terminated)
const char* data = s.data();    // Same as c_str() since C++11

// Useful when calling C library functions:
printf("%s\n", s.c_str());

c_str() returns a const char* pointing to a null-terminated copy of the string data. The pointer is valid as long as the string is not modified.

Key Takeaways

  • std::string is a dynamic, safe, and feature-rich string class from <string> that manages its own memory automatically.
  • Use getline(cin, str) to read full lines with spaces; use cin >> str to read single words.
  • size() and length() return the character count in O(1); capacity() returns allocated storage.
  • Use [] for fast access when you know the index is valid; use at() for bounds-checked access that throws on errors.
  • Concatenation works with +, +=, and append() — prefer += in loops to avoid extra copies.
  • You can compare strings directly with ==, !=, <, > — unlike C-strings.
  • find() returns the index or string::npos when not found; rfind() searches backwards.
  • substr(), replace(), erase(), and insert() let you extract and modify portions of a string.
  • Range-based for loops and iterators provide clean ways to traverse string characters.
  • Convert between strings and numbers with stoi(), stod(), stof(), and to_string(); use c_str() to obtain a const char* for C API interop.
