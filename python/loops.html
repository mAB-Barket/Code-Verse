Loops: for & while

What Are Loops?

Loops allow you to execute a block of code repeatedly, which is one of the most powerful concepts in programming. Without loops, you'd have to write the same code over and over. Python provides two main types of loops: for loops for iterating over sequences, and while loops for repeating code as long as a condition is true.

Python's for loop is different from the traditional C-style for loop. Instead of using a counter variable, Python iterates directly over items in a sequence (list, string, range, etc.). This design makes Python loops more readable, less error-prone, and more Pythonic.

Understanding loops is critical because almost every real-world program involves repetition â€” processing items in a list, reading lines from a file, waiting for user input, or running a game loop. Mastering loops will unlock the ability to write powerful, efficient programs.

The for Loop

The for loop iterates over any iterable object â€” lists, strings, tuples, dictionaries, ranges, files, and more. On each iteration, the loop variable takes on the value of the next item in the sequence:

python 
# Iterating over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}")

# Using range() for numeric loops
for i in range(5):          # 0, 1, 2, 3, 4
    print(i, end=" ")
print()  # Output: 0 1 2 3 4

for i in range(2, 10, 2):   # start=2, stop=10, step=2
    print(i, end=" ")
print()  # Output: 2 4 6 8

# Iterating over a string
for char in "Python":
    print(char, end="-")
print()  # Output: P-y-t-h-o-n-

# Iterating over a dictionary
scores = {"Alice": 95, "Bob": 87, "Charlie": 92}
for name, score in scores.items():
    print(f"{name}: {score}")

Useful Loop Functions

Python provides several built-in functions that make loops more powerful and Pythonic. These are essential tools every Python programmer should know:

python 
# enumerate() â€” get index AND value
colors = ["red", "green", "blue"]
for index, color in enumerate(colors):
    print(f"{index}: {color}")
# 0: red
# 1: green
# 2: blue

# zip() â€” iterate over multiple sequences in parallel
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]
for name, age in zip(names, ages):
    print(f"{name} is {age}")

# reversed() â€” iterate in reverse order
for i in reversed(range(5)):
    print(i, end=" ")
print()  # Output: 4 3 2 1 0

# sorted() â€” iterate in sorted order
nums = [3, 1, 4, 1, 5, 9]
for n in sorted(nums):
    print(n, end=" ")
print()  # Output: 1 1 3 4 5 9
💡

Tip: Avoid using range(len(list)) to get indices. Instead, use enumerate() â€” it's cleaner and more Pythonic. Compare: for i in range(len(items)) vs. for i, item in enumerate(items).

The while Loop

The while loop executes a block of code repeatedly as long as a given condition remains True. It's useful when you don't know in advance how many iterations are needed â€” for example, waiting for user input or processing data until a condition is met:

python 
# Basic while loop
count = 0
while count < 5:
    print(f"Count: {count}")
    count += 1

# Input validation loop
while True:
    answer = input("Enter 'yes' or 'no': ").lower()
    if answer in ("yes", "no"):
        break
    print("Invalid input! Try again.")

# Countdown
n = 10
while n > 0:
    print(n, end=" ")
    n -= 1
print("Liftoff! ðŸš€")
⚠

Warning: Be careful with while loops â€” if the condition never becomes False, you'll create an infinite loop that runs forever. Always ensure that something inside the loop changes the condition, or use break to exit. If you accidentally create an infinite loop, press Ctrl+C to stop it.

break, continue, and else

Python provides three special keywords for controlling loop flow:

  • break â€” Exit the loop immediately
  • continue â€” Skip the rest of the current iteration and move to the next
  • else â€” Execute a block if the loop completes without hitting break
python 
# break â€” exit the loop early
for num in range(100):
    if num == 5:
        print("Found 5! Stopping.")
        break
    print(num, end=" ")
# Output: 0 1 2 3 4 Found 5! Stopping.

# continue â€” skip current iteration
for num in range(10):
    if num % 2 == 0:
        continue  # Skip even numbers
    print(num, end=" ")
# Output: 1 3 5 7 9

# for...else â€” else runs if loop didn't break
def find_prime(n):
    for i in range(2, n):
        if n % i == 0:
            print(f"{n} is NOT prime (divisible by {i})")
            break
    else:
        # This runs only if the loop completed without break
        print(f"{n} IS prime!")

find_prime(7)   # 7 IS prime!
find_prime(12)  # 12 is NOT prime (divisible by 2)
📝

Note: The for...else and while...else pattern is unique to Python. The else block runs only if the loop finishes naturally (without break). It's a great tool for search operations â€” "did we find it, or not?"

Nested Loops & Performance

You can place loops inside other loops (nested loops). While powerful, be mindful of performance â€” a nested loop with two lists of size n runs nÂ² iterations. For large data sets, consider using built-in functions, list comprehensions, or libraries like NumPy instead.

python 
# Multiplication table with nested loops
for i in range(1, 6):
    for j in range(1, 6):
        print(f"{i*j:3}", end=" ")
    print()
# Output:
#   1   2   3   4   5
#   2   4   6   8  10
#   3   6   9  12  15
#   4   8  12  16  20
#   5  10  15  20  25

# Practical: find common elements
list_a = [1, 2, 3, 4, 5]
list_b = [4, 5, 6, 7, 8]

# Better: use set intersection instead of nested loops
common = set(list_a) & set(list_b)
print(f"Common elements: {common}")  # {4, 5}
